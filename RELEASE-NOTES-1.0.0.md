## Calypso 1.0.0

Initial public release.

### Included

- Offline Windows desktop application
- Research mode using PDF, DOCX, and TXT libraries
- Normal Chat mode
- Local source previews and references
- Incremental document indexing
- Document collections and summaries
- Local chat history
- Storage and cache management
- Light and dark interface themes

### Requirements

Calypso requires Ollama and the `qwen2.5:3b` model.

After installing Ollama, run:

```powershell
ollama pull qwen2.5:3b
```

### Download

Download and run `Calypso-Setup-1.0.0.exe` from the Assets section below.
