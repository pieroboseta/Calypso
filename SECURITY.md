# Security Policy

## Supported versions

Only the latest public release of Calypso is currently supported.

## Reporting a vulnerability

Please do not report security vulnerabilities through public GitHub Issues.

Use GitHub's private vulnerability reporting feature instead.

Include:

- The affected Calypso version
- Your Windows version
- Steps needed to reproduce the problem
- Any relevant logs or screenshots
- The possible impact of the issue

Please allow reasonable time for the issue to be reviewed before sharing it publicly.
